Introduction to Programing

Lesson 1: The Basics of the Web and HTML

The World Wide Web

The Web is made up of HTML documents, HTML stands for Hyper Text Mark Up Language and is the basis for virtually every web page. HTML documents are made up of Text Content (what you see), Markup (what it looks like), reference to other docs and links. Hyperlinks connects pages together. The major components of the web are:

  1. The user/browser
  2. The internet
  3. Servers
Tags

HTML elements consist of tags. These tags mostly come in pairs essentially interpret the content of the page. There is typically an opening and closing tag. There is no need to worry about remembering all tags, there countless resources available online that can be used to research tags.

Lesson 2: Creating a Structured Document with HTML

HTML, CSS and Java Script

HTML, CSS and Java have different purposes but come together to make the webpages work. HTML files control the structure, CSS controls the style (how it looks), and Java script files are the interactive components of your website

Page Structure

Elements can reside within other elements which create a "Tree like structure". Programers often refer to this stucture as the Document Object Model (DOM). This tree structure is a way to represent the hierarchical set up of the structure.

Boxes

Every element in webpages are boxes, and thinking this way is helpful for front end web developers.

Tools Programers Use

Text editors are tools that are used to edit plain text files. Scratchpad.io is a great tool for smaller projects, for more complex items Sublime Text 2 works well.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repetition is one of the most important concepts in computer programing and its essential in all areas of programing. Avoiding repetition can help you write good code, and helps programers make less mistakes.

Adding Styles

There are ways to add style without having to link to other files. The style tag can be used as well

Understanding CSS

CSS stands for Cascading Style Sheets. Specificity determins which styles are applied to an element when multiple rules could apply. A selector defines what elements of the page a style will apply to. Styles are written in the form of attribute: value;.
